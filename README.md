# 项目介绍
ApiHub 致力于降低数据开发团队在指标开发、指标运维上的难度；ApiHub 提供接口全生命周期管控并生成指标 http 接口，直连接口开发人员与前端。
ApiHub 将降低甚至取代传统数据开发时的后端角色，将传统指标、大屏开发时前后端交互提升到指标开发人员直接与前端交互。
指标将可以随时上下线或修改逻辑而不需要后端重新打包，大大降低业务响应时间
